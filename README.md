# MRF AMR Fellowship pilot code

## Description
This repository contains exemplar 'proof-of-concept' code for reconstruction the evolutionary, temporal and geogrpahic origin of AMR determinants.  Inputs are a dated phylogeny, AMR determinants and associated metadata (year and country). Further development is needed to L
* Improve efficiency
* Convert code to a user friendly R package
* Refine outputs, including visualisations
