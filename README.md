# MRF AMR Fellowship pilot code

## Description
This repository contains exemplar 'proof-of-concept' code for reconstructiing AMR dynamics.  Inputs include a phylogeny, AMR determinants and associated metadata (year and country). Further development is needed to:   
* Improve efficiency
* Convert code to a user friendly R package
* Make the code fit for purpose for intended user groups
* Refine outputs, including visualisations
* Develop documentation and tutorials
